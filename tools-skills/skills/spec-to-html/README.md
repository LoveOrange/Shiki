# spec-to-html

`spec-to-html` converts Markdown files or directories into an offline HTML
documentation site. It is generic and does not require a Shiki project.

## Usage

```bash
python tools-skills/skills/spec-to-html/scripts/publish_docs.py <input-path> --title "Spec" --fail-on-broken-links
```

Use it for human review of specs, design docs, or feature workspaces.
